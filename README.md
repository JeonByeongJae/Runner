# Runner (도망자)

> 도망가고 잡는 1vs1 추리 보드게임 — 온라인으로

**[플레이하기](https://byeongjae-jeon.github.io/Runner/)**

---

## 소개

**Runner**는 보드게임 "도망자"를 웹/모바일에서 온라인으로 즐길 수 있는 1대1 게임입니다.  
방 코드를 친구에게 공유하면 어디서든 바로 플레이할 수 있습니다.

---

## 게임 규칙

### 카드 구성
- 전체 카드: 0~42
- **0**: 출발점 (보드에 고정)
- **1, 2, 3, 42**: 도망자 시작 손패
- **4~14 / 15~29 / 30~41**: 3개 더미로 분리

### 승리 조건
| 역할 | 승리 조건 |
|------|-----------|
| 도망자 | 추격자가 모든 뒷면 카드를 맞추기 전에 **42**를 경로에 놓으면 승리 |
| 추격자 | 도망자가 42를 놓기 전에 모든 뒷면 카드를 **전부 맞춰 뒤집으면** 승리 |

### 도망자
- 카드를 **뒷면**으로 놓으며 경로를 만든다
- 놓는 카드는 직전 카드 숫자 + **1~3** 범위여야 한다
- 범위를 초과하려면 카드 밑에 **부스터 카드**를 쌓는다
  - 홀수 부스터 → +1 발자국 (범위 1 확장)
  - 짝수 부스터 → +2 발자국 (범위 2 확장)
- 매 턴 카드를 놓거나 **패스** 가능

### 추격자
- 뒷면 카드 중 원하는 것을 선택해 숫자를 추리한다
- 선택한 카드 **전부 맞아야** 뒤집힘 — 하나라도 틀리면 아무것도 뒤집지 않는다
- 추리 성공 시 해당 카드 + 밑의 부스터 카드까지 모두 공개

### 턴 진행

**첫 턴:**
| 역할 | 드로우 | 행동 |
|------|--------|------|
| 도망자 | 4~14에서 3장 + 15~29에서 2장 | 카드 최대 **2장** 뒷면으로 놓기 |
| 추격자 | 원하는 더미에서 2장 | 즉시 추리 가능 |

**이후 턴 (번갈아 반복):**
- 도망자: 원하는 더미에서 1장 드로우 → 카드 놓기 or 패스
- 추격자: 원하는 더미에서 1장 드로우 → 추리

## 기술 스택

- **Frontend:** React + TypeScript + Vite
- **실시간 동기화:** Firebase Realtime Database
- **배포:** GitHub Pages

---

## 개발 시작하기

### 요구사항
- Node.js 18+
- Firebase 프로젝트 (Realtime Database 활성화)

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/byeongjae-jeon/Runner.git
cd Runner

# 의존성 설치
npm install

# 환경변수 설정
cp .env.example .env.local
# .env.local에 Firebase 설정 입력

# 개발 서버 실행
npm run dev
```

### 빌드 및 배포

```bash
# 프로덕션 빌드
npm run build

# GitHub Pages 배포 (CI 자동 배포)
# main 브랜치 push 시 GitHub Actions가 자동으로 배포
```
